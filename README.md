This python program is supposed to let the user draw digits (0 - 9) using tkinter and let the model guess what the digit is.
The model is a CNN and is trained on keras' mnist dataset.
This project is mostly for fun and as a capstone project for one of my AI/ML classes.

Run the main.py file and draw a digit on the tkinter interface. Press predict to call the model and output a prediction. You can erase your drawings using the Erase button (it erases everything). It is recommended to use the default window dimensions because adjusting it can affect the model's accuracy.
