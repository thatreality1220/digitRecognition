This python program is supposed to let the user draw digits (0 - 9) using tkinter and let the model guess what the digit is.
The model is a CNN and is trained on keras' mnist dataset.
This project is mostly for fun and as a capstone project for one of my AI/ML classes.

First, run the model.py file to train the model. Next, run main.py and draw a digit on the drawing interface. Once you are done, press Predict to call the model and output a predicted digit. Press Erase when you want to erase everything on the screen. It is recommended to use the default window dimensions because adjusting it can affect the model's accuracy.
